# 搞怪语音盒 - 用户安装教程

## ⚠️ 安全提示说明

安装时可能会提示"**风险应用**"或"**诈骗应用**"，这是正常的！

**原因：**
- 本应用未上架应用商店
- 系统对非商店应用的安全提示
- **不是真的诈骗应用，请放心安装**

---

## 📱 Android / 鸿蒙 安装步骤

### 步骤1：下载 APK
- 扫描二维码 或 点击下载链接
- 下载 `app-release.apk` 文件

### 步骤2：允许安装
看到提示时，按以下操作：

**提示："此应用可能存在风险"**
→ 点击"**仍要安装**"或"**继续安装**"

**提示："未知来源"**
→ 点击"**设置**" → 开启"**允许此来源**" → 返回继续安装

**提示："纯净模式"（鸿蒙）**
→ 点击"**继续安装**" → 输入锁屏密码 → 确认安装

### 步骤3：完成安装
- 点击"**安装**"
- 等待安装完成
- 点击"**打开**"

---

## 🔧 如果无法安装

### 方法1：关闭纯净模式（鸿蒙）
1. 设置 → 系统 → 纯净模式
2. 关闭纯净模式
3. 重新安装 APK

### 方法2：允许未知来源（Android）
1. 设置 → 安全
2. 找到"未知来源"或"安装外部应用"
3. 找到"文件管理"或"浏览器"
4. 开启"允许安装"
5. 重新安装 APK

## ⚙️ 首次使用设置（重要）

安装后需要开启权限，才能在微信录音时使用：

### 1️⃣ 开启悬浮窗权限
**路径：** 设置 → 应用 → 搞怪语音盒 → 权限 → 悬浮窗
- ✅ 开启"悬浮窗"

### 2️⃣ 开启后台运行
**路径：** 设置 → 应用 → 搞怪语音盒 → 权限 → 后台运行
- ✅ 开启"允许后台运行"

### 3️⃣ 关闭省电限制
**路径：** 设置 → 电池 → 应用耗电管理 → 搞怪语音盒
- ✅ 选择"允许"或"无限制"

### 4️⃣ 锁定后台（推荐）
**操作：** 打开多任务 → 找到"搞怪语音盒" → 下拉锁定
- 🔒 防止被系统清理

## 🎯 使用技巧

### 在微信中使用：

**方法1：分屏模式（最推荐）**
1. 打开微信聊天
2. 从屏幕侧边滑出多任务
3. 选择"搞怪语音盒"
4. 进入分屏
5. 同时操作两个应用

**方法2：循环播放**
1. 打开"搞怪语音盒"
2. 开启"循环播放"
3. 点击语音播放
4. 切换到微信录音

**方法3：快速切换**
1. 打开"搞怪语音盒"
2. 点击语音播放
3. 快速切换到微信
4. 按住录音

## 📦 导入语音包

### 从网盘下载：
1. 点击"下载语音包"
2. 选择"夸克网盘"或"百度网盘"
3. 下载 ZIP 压缩包到手机
4. 返回 App，点击"导入语音包"
5. 选择下载的 ZIP 文件
6. 自动解压并导入

### 从电脑传输：
1. 电脑上准备 ZIP 压缩包
2. 通过数据线/蓝牙/微信传到手机
3. 保存到"文件管理"
4. 在 App 中点击"导入语音包"
5. 选择 ZIP 文件

## ❓ 常见问题

### Q: 为什么提示"诈骗应用"？
**A:** 因为未上架应用商店，系统的安全提示。点击"继续安装"即可，应用是安全的。

### Q: 切换到微信后声音停了？
**A:** 需要开启"悬浮窗"和"后台运行"权限。

### Q: 声音断断续续？
**A:** 关闭省电模式，并在多任务界面锁定应用。

### Q: 微信录不到声音？
**A:** 
- 确保手机音量已开启
- 使用分屏模式
- 或使用循环播放

### Q: 怎么删除语音？
**A:** 长按语音按钮即可删除（内置语音除外）。

### Q: 怎么批量删除？
**A:** 点击分类右侧的"编辑"按钮，多选后删除。
