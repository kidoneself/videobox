# 🎙️ 语音盒

## ✨ 特性

- 📦 ZIP批量导入 - 一键导入整个语音包
- 🗂️ 自动分类 - ZIP文件名即分类名
- 💾 本地存储 - 数据永久保存
- 🎵 内置示例 - 3个默认语音
- ⚡ 完全离线 - 无需网络

## 🚀 使用方法

### 导入ZIP语音包

1. 准备ZIP文件（例如：`经典语录.zip`）
2. ZIP内放入音频文件（MP3/M4A/WAV/AAC）
3. 点击"📦 导入ZIP"
4. 选择文件，自动导入
5. 完成！自动按ZIP文件名分类

### 删除语音

长按语音按钮即可删除

## 📦 ZIP打包示例

```
经典语录.zip
├── 我要验牌.mp3
├── 给我擦皮鞋.mp3
└── 牌没有问题.mp3
```

**导入后效果：**
- 分类名："经典语录"
- 3个语音按钮（文件名即按钮文字）

## 💡 推荐使用方式

1. 制作多个主题ZIP包
2. 上传到网盘分享
3. 用户下载后一键导入
4. 完美！

## 🎯 支持格式

- MP3
- M4A
- WAV
- AAC
