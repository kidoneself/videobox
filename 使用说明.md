# 🎙️ 语音盒使用说明

## 📦 批量导入语音（推荐）

### 🎯 重要：一个压缩包 = 一个分类

**ZIP文件名会成为分类名！**

例如：
- `经典语录.zip` → 分类名："经典语录"
- `笑声.zip` → 分类名："笑声"
- `星爷语录.zip` → 分类名："星爷语录"

---

### 步骤 1: 准备语音包

#### 示例 1: 经典语录包
```
经典语录.zip
├── 我要验牌.mp3
├── 给我擦皮鞋.mp3
├── 牌没有问题.mp3
└── 你作弊.mp3
```
导入后显示为：
- 分类："经典语录"
- 语音：我要验牌、给我擦皮鞋、牌没有问题、你作弊

#### 示例 2: 笑声包
```
笑声.zip
├── 哈哈哈.mp3
├── 嘿嘿嘿.mp3
└── 嘻嘻嘻.mp3
```
导入后显示为：
- 分类："笑声"
- 语音：哈哈哈、嘿嘿嘿、嘻嘻嘻

---

### 步骤 2: 上传分享

1. 把 ZIP 上传到网盘（百度网盘、阿里云盘等）
2. 分享链接给用户
3. 用户下载到手机

---

### 步骤 3: 导入

1. 打开语音盒应用
2. 点击 **"📦 导入ZIP"**
3. 选择下载的 ZIP
4. 完成！自动按分类显示

---

## 📁 推荐的打包方式

### 方式 1: 按主题分包（推荐）
```
经典语录.zip
笑声.zip
角色台词.zip
搞笑音效.zip
```
用户可以只下载自己想要的包

### 方式 2: 完整包
```
星爷语音大全.zip
├── 我要验牌.mp3
├── 给我擦皮鞋.mp3
├── 哈哈哈.mp3
├── ... (50个文件)
```
所有语音一个包，分类名就是"星爷语音大全"

## 🎯 命名技巧

### ZIP文件命名
- ✅ `经典语录.zip` - 简洁明了
- ✅ `星爷搞笑语录.zip` - 有特色
- ✅ `赌神系列.zip` - 按电影分
- ❌ `archive.zip` - 太随意
- ❌ `新建文件夹.zip` - 没意义

### 音频文件命名
- ✅ `我要验牌.mp3` - 清晰准确
- ✅ `给我擦皮鞋.mp3` - 完整台词
- ❌ `1.mp3` - 看不懂
- ❌ `audio_001.mp3` - 没意义

## ➕ 单个添加语音

如果只想添加1-2个语音：

1. 点击 **"+ 添加语音"**
2. 输入文字
3. 选择分类（经典语录、笑声、角色、其他）
4. 选择音频文件
5. 确定

---

## 🗑️ 管理语音

- **删除**: 长按语音按钮
- **清理缓存**: 点击底部"清理缓存"

## 🎨 实际使用示例

### 场景 1: 制作星爷语音包
```
1. 收集整理音频
   经典语录/
   ├── 我要验牌.mp3
   ├── 给我擦皮鞋.mp3
   └── 牌没有问题.mp3
   
   笑声/
   ├── 哈哈哈.mp3
   └── 嘿嘿嘿.mp3

2. 分别压缩
   经典语录.zip
   笑声.zip

3. 上传网盘，分享链接

4. 用户下载后：
   - 导入 经典语录.zip → 自动归类到"经典语录"
   - 导入 笑声.zip → 自动归类到"笑声"
```

### 场景 2: 个人收藏
```
我的收藏.zip
├── 音效1.mp3
├── 音效2.mp3
└── 音效3.mp3

导入后全部在"我的收藏"分类下
```

## 💡 专业建议

### 对于内容制作者：
1. **分类明确** - 一个主题一个ZIP
2. **命名规范** - 文件名清晰易懂
3. **质量控制** - 统一音量，裁剪杂音
4. **合理分包** - 每包10-30个语音最佳
5. **提供说明** - 告诉用户有哪些分类

### 对于用户：
1. **按需下载** - 只下载喜欢的分类
2. **保存ZIP** - 方便重装后恢复
3. **分享朋友** - 好东西一起用
