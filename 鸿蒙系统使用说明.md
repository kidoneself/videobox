# 鸿蒙系统使用说明

## ⚠️ 重要权限设置

鸿蒙系统对音频管理比较严格，需要开启以下权限才能在微信录音时播放语音：

### 1️⃣ 开启悬浮窗权限（必须）
**路径：** 设置 → 应用 → 搞怪语音盒 → 权限管理 → 悬浮窗
- ✅ 开启"悬浮窗"

### 2️⃣ 开启后台运行权限（必须）
**路径：** 设置 → 应用 → 搞怪语音盒 → 权限管理 → 后台运行
- ✅ 开启"允许后台运行"

### 3️⃣ 关闭省电模式（推荐）
**路径：** 设置 → 电池 → 应用耗电管理 → 搞怪语音盒
- ✅ 选择"允许"或"无限制"

### 4️⃣ 锁定后台（推荐）
**操作：** 打开多任务界面 → 找到"搞怪语音盒" → 下拉锁定
- 🔒 防止系统自动清理

## 🎯 使用技巧

### 在微信语音通话中使用：

**方法1：先播放，再录音**
1. 打开"搞怪语音盒"
2. 点击要播放的语音
3. 快速切换到微信
4. 按住录音按钮
5. 语音会继续播放并被录制

**方法2：使用循环播放**
1. 打开"搞怪语音盒"
2. 开启"循环播放"开关
3. 点击要播放的语音
4. 切换到微信录音
5. 语音会持续播放

**方法3：使用分屏（最推荐）**
1. 打开微信对话框
2. 从屏幕侧边滑出多任务
3. 选择"搞怪语音盒"
4. 进入分屏模式
5. 同时操作两个应用

## ❓ 常见问题

### Q: 切换到微信后，声音停止了？
**A:** 检查以下设置：
- 悬浮窗权限是否开启
- 后台运行权限是否开启
- 是否被省电模式限制

### Q: 声音断断续续？
**A:** 
- 关闭省电模式
- 在多任务界面锁定应用
- 清理后台其他应用释放内存

### Q: 微信录不到声音？
**A:**
- 确保手机音量已开启
- 尝试使用扬声器模式
- 使用分屏模式
